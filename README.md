# 📈 Stocker – Cloud-Based Stock Trading Web Application

### 🚀 Project Overview

**Stocker** is a cloud-based stock trading web application developed using **Flask** and **AWS Cloud Services**. The platform enables users to securely manage their investment portfolios by buying, selling, and tracking stocks through an intuitive web interface. Designed with a scalable cloud architecture, the application integrates AWS services for secure storage, efficient data management, and reliable deployment.

The application features user authentication, portfolio management, stock transaction processing, and RESTful APIs to provide a seamless trading experience. It demonstrates the practical implementation of cloud computing concepts and full-stack web development using Flask and AWS.

### ✨ Key Features

* Secure user authentication and authorization
* Buy and sell stocks with real-time portfolio updates
* Portfolio management and investment tracking
* RESTful API integration for stock operations
* Cloud-based deployment using AWS EC2
* Secure file storage using Amazon S3
* Fast and scalable database management with Amazon DynamoDB
* Responsive and user-friendly web interface

### 🎯 Project Objective

The objective of **Stocker** is to provide a secure, scalable, and cloud-enabled stock trading platform where users can manage their investments, execute stock transactions, and monitor their portfolios while gaining practical experience in Flask development, REST API integration, and AWS cloud deployment.
